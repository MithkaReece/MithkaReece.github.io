Rogue-Like

Previous Development - Summer 2022

Initially a group project, set out to make a basic Rogue-Like Game and learn game development skills. Due to many of the members having other commitments most of them did not have for any contributions. One member draw a few pieces of concept art and another coded a brief skeleton of a stats system. These contributions were very minor and with no intention of any other members to continue this project and the majority of the work up to this stage coming from myself, I decided to turn this into my own solo project. This allows me to further refine my game development skills and getting the game closer to a finish state will allow me to utilising finishing technologies such as particles effects, visual effects, optimisations and compatibity of devices that can run the game.

Current Development - Summer 2023

My goal at the moment is to complete the most basic of systems such as combat, procedure level generation and basic progression system to round off the game play loop. I then want to finish off the game in that state but still allowing any new content to be added when needed.
