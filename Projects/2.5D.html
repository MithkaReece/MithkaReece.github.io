Development - September-Current 2023

I first setup simple movement and animations for the player which from previous projects was a lot more straightforward. I use a basic Rigidbody2D for collisions when not airboard as no 2.5D aspects are needed. My custom physics, make Rigidbody2D kinematic whilst midair so I can handle the collisions, I prevent movement towards objects on collision. I keep track of the ground position of the player and check if this collides with a wall, if so I compare the player's height with the wall height saved within the tilemap.

The height levels of all tiles are pre-generated for the level. This is done by traversing all the ground tiles and change level based on stairs. The wall heights are then calculated based on the first ground level that appear above it.
