Asteroids C++

Development - July-Current 2023

To reinforce my C++ skills and to learn ECS I created Asteroids again. I used entt ECS library and sfml for visuals. This involved creating a way to manipulate and traverse different game scenes achieved by creating a scene manager. This scene manager involves tracking scenes which track entities and systems. Scenes can be created, deleted or paused which affect the entities and systems of the scene. A separate system manager controls the actual creation, deletion and updating of the systems in use. Systems, scene manager and system manager can communciate through events using entt event dispatcher. The rest of the Asteroid game is also recreated using this new structure and plays like my previous JS version. The additional content consists of a main menu, pause menu and saving of high score. Later I wish to polish the game using particle and sound effects as well as optimisations and visual improvements.
