Cooking Game Farming Game

Unity's Junior Pathway Course

Summer 2023

In this course, I learned how to create a cooking game in Unity, which provided me with a comprehensive understanding of the entire game development process, from inception to completion. Moreover, I acquired valuable insights into the implementation of improved and more organized coding standards suitable for larger game code bases.

Key Features Learnt

  • Reinforce Game Dev Foundations: This course helped me gain confidence in using an engine like Unity by guiding me through the creation of several small games. This practical experience enabled me to become more efficient in setting up processes that many games share, such as handling UI, updating stats, managing scene transitions, and establishing general gameplay loops.

Challenges, solutions and lessons learnt

The combination of step-by-step tutorials followed by independent tasks allowed me to develop confidence in my abilities and become more familiar with using a game engine. One aspect that presented a challenge, as it was a new process for me, was the theoretical information provided about creating documentation for different game features. Typically, I would use my own Trello board to outline the features I wanted with a rough sense of their priority. However, this course demonstrated that you can delve into much greater detail regarding the importance of each feature and estimating the effort required to implement them.

Below, you will find a collection of games produced during this course, all of which have been built for the web. The prototypes are games that follow step-by-step tutorials, however, they often include optional extension features which I implemented myself. The challenges encompass either complete games with numerous bugs that needed fixing or the beginnings of a game with many features for me to add myself.
